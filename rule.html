📗 サーバー規約について

  • ✅ 運営の指示に従ってください
  • ✅ 運営判断で処罰する場合もあります
  • ✅ 禁止行為を行うと処罰が行われます

📕 禁止行為とされていること

  • 🚫 迷惑行為
  • 🚫 不正行為
  • 🚫 宣伝行為
  • 🚫 相手を不快にする行為
  • 🚫 サーバーを攻撃する行為

🪄 サーバー参加ルール

  • 🚀 アクセスを繰り返さないことバグ・回線落ちなどはOK。サーバーに無駄な負荷がかかるため。
  • 🚀 不公平なMODの禁止軽量化やレシピ・MAP・空腹はOK。不安なMODは質問してください。
  • 🚀 マクロ・ツール・自動操作の禁止使用できない機種と不平が生じるため。種類によっては負荷がかかるため。
  • 🚀 サブ垢は1アカウントまで別の人がプレイするなどはOK。サーバーに無駄な負荷がかかるため。
  • 🚀 サーバー内AFK回避の禁止AFK設定している意味がなくなるため。自動再接続もAFK回避とみなします。

⛏️ サーバー負荷ルール

  • 📖 長時間チャンク読み込みの禁止目的ありのチャンク読み込みはOK。長時間は目安5分以上。
  • 📖 MOBの大量使用の禁止大量のMOBがいるところに放置、大量の確保・捕獲等（目安は10体以上）。
  • 📖 大きい負荷がかかる装置の禁止大量演算、常時稼働、初期リス付近などの装置は禁止。

🔊 チャットルール

  • 💬 あいさつマナーあいさつはきちんと返しましょう。自分からもあいさつをしよう。
  • 💬 迷惑なチャットの禁止同じ文字の連打（例：aaaa）、スパム、クレクレ、暴言、不満など。
  • 💬 チャットでの会話等ルール身内ノリ等は /tell や /ch で。公開チャットでは適切に。過度な独り言は控えてください。
